### Sistema de estudiantes 
Este proyecto es un sistema que contiene diferentes funciones prácticas como agregar estudiantes con datos como nombre, edad, grado, materias favoritas y promedio. 
También permite mostrar, buscar, actualizar y eliminar estudiantes. Puede calcular el promedio general del grupo. Y contiene un menú interactivo.
